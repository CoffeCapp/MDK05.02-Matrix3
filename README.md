# Калькулятор матриц

 Консольное приложение на языке Go для выполнения операций над матрицами 2*2, 3*3, 4*4. Представляет собой интерактивное меню, для создания матриц и выполнения математических операций над ними.

## Возможности данной программы

Данная программа позволяет:

1. Ввести две матрицы
2. Умножить первую матрицу на число
3. Умножить вторую матрицу на число
4. Перемножить матрицы между собой
5. Сложить матрицы.

## Как работает программа?

1. Запуск main, объявление в ней переменных matrix1, matrix2, size, exists1, exists2.
2. Запускается бесконечный цикл for. 
3. Выводится меню действий.
4. Пользователь вводит номер действия(choice) и программа обрабатывает его через switch.
5. Ввод матрицы (пункты 1 и 2):
---
 1. case 1: (Ввести первую матрицу):
 2. fmt.Scan(&size) - Считывает размер матрицы (2, 3 или 4).
 3. if size < 2 || size > 4 - Проверяет, допустим ли размер.
 4. matrix1 = getMatr(size) - Вызывает функцию getMatr, которая:
 5. Создает пустую матрицу заданного size.
 6. Циклами for запрашивает у пользователя каждый элемент [i][j].
 7. fmt.Scan(&matrix[i][j]) - Считывает введенный элемент.
 8. if err != nil - Проверяет, ввел ли пользователь корректное число. Если нет, выводит ошибку и запрашивает снова.
 9. exists1 = true - Устанавливает флаг, что первая матрица существует.
 10. Циклами for проходит по строкам и столбцам матрицы.
 11. fmt.Print(val, " ") - Печатает элементы в строку, разделяя пробелом.
 12. fmt.Println() - Переходит к следующей строке после каждой строки матрицы.
 13. case 2. Всё тоже самое, что и в case 1.
---
6. case 3 выводит матрицы.
7. case 4 и 5.
---
 1. fmt.Scan(&num) - Считывает число, на которое нужно умножить.
 2. result := matrMul(matrix1, num) - Вызывает функцию matrMul, которая:
 3. Создает новую матрицу result такого же размера, как matrix1.
 4. Циклами for проходит по всем элементам.
 5. result[i][j] = matrix[i][j] * num - Каждый элемент новой матрицы равен соответствующему элементу старой, умноженному на num
 6. Возвращает новую матрицу result.
 7. showMatr(result) - Печатает результат умножения.
 8. showMatr(matrix1) или (matrix2) - Вызывает функцию showMatr, которая:
---
8. case 6 умножение матриц друг на друга.
---
 1. Получает размеры rowsA, colsA (кол-во столбцов первой матрицы), colsB (кол-во столбцов второй).
 2. Создает результирующую матрицу result размером rowsA x colsB.
 3. i и j - индексы для результирующей матрицы.
 4. k - индекс для выполнения суммы произведений (строка a * столбец b).
 5. result[i][j] += a[i][k] * b[k][j] - Выполняет основной расчет элемента результата.
 6. Возвращает result.
 7. showMatr(result) - Печатает результат умножения матриц.
---
9. case 7 выход из программы.

## Используемые функции

1. main() — Она содержит основной цикл, который показывает меню и обрабатывает выбор пользователя.
2. showMatr(matrix [][]int) — Принимает матрицу и печатает её на экран в виде таблицы, разделяя элементы пробелами.
3. matrMul(matrix [][]int, num int) [][]int — ринимает матрицу и число. Умножает каждый элемент этой матрицы на заданное число и возвращает новую матрицу с результатом
4. matrMul2(matrix [][]int, num int) [][]int — Делает то же самое, что и matrMul, просто умножает матрицу на число. Он просто вызывает matrMul внутри себя
5. multiplyMatrices(a, b [][]int) [][]int — Принимает две матрицы a и b. Выполняет математическое умножение этих двух матриц и возвращает результирующую матрицу.
6. Запрашивает у пользователя ввод элементов для квадратной матрицы заданного size (2, 3 или 4). Создаёт и заполняет матрицу значениями, введёнными пользователем, и возвращает её.
